# 개인정보처리방침

## 1. 개인정보의 수집 및 이용
이 앱("WayList")은 사용자의 개인정보를 수집하지 않습니다.

다만, 앱의 주요 기능 제공을 위해 기기 위치 정보(Location)에 접근할 수 있습니다.  
위치 정보는 다음 목적에 한해 사용됩니다.

- 사용자가 저장한 장소를 지도에 표시하기 위해
- 사용자의 현재 위치를 지도에 표시하기 위해
- 위치 기반 알림 기능을 제공하기 위해

위치 정보는 앱 내 기능 제공을 위해서만 일시적으로 사용되며,  
서버에 저장되거나 외부로 전송되지 않습니다.

## 2. 제3자 제공
이 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다.

## 3. 데이터 보안
이 앱은 별도의 사용자 데이터를 수집하지 않으므로, 데이터 보안과 관련된 조치를 취할 필요가 없습니다.

## 4. 외부 서비스 및 광고
이 앱은 외부 서비스(예: Google Analytics, 광고 SDK 등)를 사용하지 않습니다.

## 5. 변경 사항
이 개인정보처리방침은 변경될 수 있으며, 변경 사항은 앱 내 공지될 예정입니다.
